
# Canban board

Vue.js 3, Pinia, TailwindCSS

## Run Locally

Clone the project

```bash
  git clone https://github.com/Divrun/kanban-board
```

Go to the project directory

```bash
  cd kanban-board
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run rev
```
